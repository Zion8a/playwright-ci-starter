# Portfolio – playwright-ci-starter

## Vad detta repo visar
- UI-testautomation med Playwright
- CI i GitHub Actions (push/PR)
- Cross-browser-körning (Chromium + Firefox)
- Artifacts för felsökning: HTML report + test-results (screenshot/video vid fail)

## Snabb demo
1) Push en ändring → CI triggas
2) Vid fail: ladda ner artifacts och öppna playwright-report/index.html
